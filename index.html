Using the SDE Solver

1. Equation Input

Enter your stochastic differential equation using mathematical notation:

  • Drift coefficient: The deterministic part (dt term)
  • Diffusion coefficient: The stochastic part (dW term)
  • Use standard mathematical functions: sin, cos, exp, log, sqrt
  • Parameters should be entered as JSON in the parameters field

2. Equation Types

Itô SDE

Standard formulation using Itô calculus. Follows the Itô isometry and has zero expected value for stochastic integrals.

Stratonovich SDE

Uses midpoint evaluation. Follows classical chain rule and is more intuitive for some physical applications.

3. Stochastic Process Types

The solver supports various stochastic processes:

  • Geometric Brownian Motion: For stock prices and financial models
  • Ornstein-Uhlenbeck Process: Mean-reverting process
  • Vasicek Model: Interest rate modeling
  • CIR Model: Cox-Ingersoll-Ross model
  • Standard Brownian Motion: Standard Wiener process
  • Exponential Martingale: For risk-neutral pricing
  • Poisson Process: Jump processes
  • Jump Diffusion: Processes combining diffusion and jumps

4. Simulation Parameters

Configure the simulation to visualize the behavior of your SDE:

  • Time range: Define the simulation time interval
  • Number of trajectories: How many sample paths to generate
  • Initial condition: Starting value for the process

About

SDE Symbolic Solver

This web application provides symbolic solutions and simulations for Stochastic Differential Equations (SDEs). It implements methods based on the research of Chu-Ching Huang (2011) and extends them with modern web technologies.

Reference: Huang, C. C. (2011). Python Solver for Stochastic Differential Equations. Matimyas Matematika, 34(1-2), 87-97.

Features

  • Symbolic solution of Itô and Stratonovich SDEs
  • Step-by-step solution derivation
  • Numerical simulation with customizable parameters
  • High-quality plots with publication-ready styling
  • Support for multiple stochastic processes
  • Export results to PDF and LaTeX

Technical Details

  • Backend: Python with FastAPI, SymPy for symbolic computation
  • Frontend: HTML, CSS, JavaScript with Tailwind CSS
  • Visualization: Plotly for interactive plots
  • Mathematical Typesetting: MathJax for LaTeX rendering
